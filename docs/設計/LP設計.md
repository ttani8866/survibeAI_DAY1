# AIプログラミング学習サービス LP設計書

## 概要

SurviveAI - AIがあなたのコードを24時間レビューし、プロフェッショナルなコーディングスキルを身につけるための学習サービス

---

## ページ構成

### 1. ヒーローセクション

#### デザイン
- フルスクリーン表示（100vh）
- ダークモードベース（#0a0a0a）
- グラデーション背景エフェクト
- framer-motionによる文字アニメーション

#### コンテンツ
| 要素 | 内容 |
|------|------|
| サブタイトル | WE ARE A |
| メインタイトル | CODE INNOVATOR |
| キャッチコピー | AIがあなたのコードを24時間レビュー。プロフェッショナルなコードを書くための、新しいスタンダードを創造する。 |
| CTAボタン（プライマリ） | 無料で始める → /onboarding/signup |
| CTAボタン（セカンダリ） | お問い合わせ → /contact |

---

### 2. サービス特徴セクション

#### 特徴カード 3枚

| No. | タイトル | 説明 |
|-----|----------|------|
| 01 | AI Code Review | AIが改善点を指摘し、より良い書き方を提案。プロのようなコードを書けるようになる。 |
| 02 | Learning Path | 初心者にも分かりやすい解説で、ステップバイステップで学習できる。 |
| 03 | 24/7 Support | 困った瞬間にすぐ質問して解決。24時間いつでもAIがサポート。 |

#### デザイン
- 3カラムグリッドレイアウト（レスポンシブ対応）
- ホバー時にグロー効果
- 左ボーダーラインでアクセント
- ナンバリングはグラデーション表示

---

### 3. 使い方ステップセクション

#### ステップ構成

| Step | タイトル | 説明 | アイコン |
|------|----------|------|----------|
| 01 | アカウント作成 | メールアドレスまたはGoogleアカウントで簡単登録。30秒で完了します。 | PersonAdd |
| 02 | コードを書く | エディタでコードを書くと、AIがリアルタイムで分析を開始します。 | Code |
| 03 | フィードバック確認 | AIからの具体的な改善提案を確認。なぜそうすべきかも丁寧に解説。 | Lightbulb |
| 04 | スキルアップ | 繰り返し学習することで、プロレベルのコーディングスキルを習得。 | TrendingUp |

#### デザイン
- タイムライン形式の縦レイアウト
- ステップ間を接続線で繋ぐ
- 各ステップはスクロールアニメーションで表示
- アイコンはMUI Iconsを使用

---

### 4. CTAセクション

#### デザイン
- 中央寄せレイアウト
- 背景にグロー効果
- シマーアニメーション付きテキスト

#### コンテンツ
| 要素 | 内容 |
|------|------|
| ヘッドライン | Ready to innovate? |
| サブテキスト | 今すぐ無料で始めて、コーディングスキルを次のレベルへ。 |
| CTAボタン | 無料で始める → /onboarding/signup |

---

### 5. フッターセクション

#### コンテンツ
| 要素 | 内容 |
|------|------|
| コピーライト | © 2025 SurviveAI. All Rights Reserved. |
| リンク | Contact → /contact |

#### デザイン
- 上部にボーダーライン
- コピーライトとリンクを横並び配置
- レスポンシブでラップ対応

## 技術仕様

### 使用技術
- **フレームワーク**: Next.js 16 + React 19
- **UIライブラリ**: MUI (Material-UI) v7
- **アニメーション**: framer-motion
- **スタイリング**: MUI sx props + Emotion

### レスポンシブ対応

| ブレークポイント | カラム数（特徴カード） | タイポグラフィ調整 |
|------------------|------------------------|-------------------|
| xs (0-599px) | 1カラム | fontSize減少 |
| md (900px+) | 3カラム | 標準サイズ |

### ダークモード対応

| 要素 | 色設定 |
|------|--------|
| 背景 | #0a0a0a |
| テキスト（メイン） | #ffffff |
| テキスト（セカンダリ） | rgba(255,255,255,0.5) |
| アクセント | #6366f1 (Indigo) |
| ボーダー | rgba(255,255,255,0.1) |
